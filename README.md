# Tetris-in-C

Only on windows.

Compile using linker options: `-lgdi32 -lwinmm`

## Controls
### Menu
`WASD` or arrow keys to move
### In-Game
`A, LEFT`  - Move left <br>
`D, RIGHT` - Move right <br>
`W, UP, I` - Rotate left <br>
`J`        - Rotate right <br>
`M` - Mute music <br>
`N` - Mute all music and sounds <br>
`C` - Disable colors <br>
`RETURN, ESCAPE, P` - Pause
